My 1st Nginx Page - Created by Bash Scripting - v1.2

This was created by a local workstation script file which:

  1. SSH'd into the remote machine.
  2. Set safety options in the bash environment.
  3. Created deploy user and copied ssh key to their home.
  4. Installed Nginx.
  5. Securely copied an already built index.html file into the directory Nginx serves templates from, i.e., /var/www/html/...
  6. Rebooted Nginx so that it would serve the index.html file.
