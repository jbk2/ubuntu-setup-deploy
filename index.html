My 1st Nginx Page - Created by Bash Scripting

This was created by a local workstation script file which:

  1. SSH'd into the remote machine.
  2. Set safety options in the bash environment.
  3. Installed Nginx.
  4. Securely copied an already built index.html file into the directory Nginx serves templates from, i.e., /var/www/html/...
  5. Rebooted Nginx so that it would serve the index.html file.
